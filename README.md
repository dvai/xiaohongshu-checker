# 小红书违规词检测工具

这是一个零依赖的检测小红书正文里的高风险词、导流信息、夸大承诺和常见敏感表述的本地网页工具。
直接打开 [index.html](./index.html) 即可使用，或者你也可以部署到自己的服务器上使用。
在线预览：[小红书违规词检测工具](https://dvai.github.io/xiaohongshu-checker/)。

## 使用方式

1. 直接双击打开 `index.html`，或者在当前目录运行 `python3 -m http.server 8000` 后访问 `http://localhost:8000`。
2. 把你准备发布的小红书正文粘贴进去。
3. 点击“立即检测”，查看高亮预览和命中详情。
4. 如果你有自己的行业敏感词，可以在“自定义词库”里每行补一个词并保存。

## 当前能力

- 内置基础词库：绝对化用语、医疗功效、收益承诺、导流联系方式、跨平台引流、高仿灰产、违法违禁、低俗擦边、迷信玄学。
- 支持关键词命中和正则规则命中。
- 支持浏览器本地保存自定义词库。
- 支持原文高亮、左右对照和命中详情展示。
